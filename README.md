# proyecto_php_2026
##Descripción
Contiene el código fuente en el lenguaje PHP de la imagen con el mensaje de "Hola mundo" y el texto "Hola, soy José Alejandro", usando elementos HTML y CSS para los estilos.
##Prerrequisitos
Como parte de los requeriminetos a instalar para su correcta ejecución se listan las siguientes.
- Sistema operativo (Windows/Linux)
- Visual Studio Code
- Apache2
- Lenguaje de programación PHP
##Pasos para su Ejecución
- Iniciar la máquina con el sistema operativo que contiene el servidor web apache2
- Ejecutar la terminal del servidor
- Escribir el comando "ifconfig" para visualizar la direccción
- Conectarse al servidor mediante la línea de comandos de la máquina que tiene Visual Studio Code
- Abrir un navegador web
- Ingresar la dirección IP del servidor en la URL
- Visualizar el contenido descrito en la sección de Descripción
